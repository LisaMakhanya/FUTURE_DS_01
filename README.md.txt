Superstore Sales Dashboard (Excel)

 📊 Project Overview
This project analyzes Superstore sales data using Excel to uncover insights related to sales performance, profitability, customer behavior, and regional trends.

An interactive dashboard was built to support data-driven decision-making.

---


🔍 Key Insights
- Technology is the most profitable category, contributing the highest overall profit.
- Furniture generates strong sales but has significantly lower profit margins.
- Office Supplies provide stable and consistent profitability.
- The West region outperforms other regions in both sales and profit.
- Sales show clear seasonality, with peak performance toward the end of the year.
- Certain sub-categories generate high revenue but low or negative profit, indicating pricing inefficiencies.

---

 🛠 Tools & Techniques
- Microsoft Excel
- Data Cleaning & Validation
- Exploratory Data Analysis (EDA)
- PivotTables & Pivot Charts
- Slicers & Timelines
- Interactive Dashboard Design
